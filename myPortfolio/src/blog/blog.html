Starting My Journey: Understanding GIT

What is GIT? GIT is a version control system that allows developers to manage source code with reduced risk and track changes. While coding, developers using GIT use a system called branches to create isolated copies of the source code from a “master” branch. Once branches are created, they can be pulled and edited by many different users. If the changes they make are desirable, they can commit these changes and push it back to a centralized location on GitHub. If the changes are not desirable, then only the copy of the master branch is affected, and the original code is undisturbed. This ensures that edits are made without losing any progress.
